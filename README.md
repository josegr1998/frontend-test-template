#PROJECT INFO

TO RUN THE PROJECT USE THE FOLLOWING COMMANDS:

npm install

npm run dev

TO CREATE A BUILD USE THE FOLLOWING COMMANDS:

npm run build

npm run start

TO RUN THE TEST CASES USE THE FOLLOWING COMMAND:

npm run test
